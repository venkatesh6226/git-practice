HTML Lists

1. Ordered List (Numbered List)

An ordered list is used when the order of items matters (e.g., steps in a process).

  1. Step 1
  2. Step 2
  3. Step 3

2. Unordered List (Bullet Points)

An unordered list is used when the order of items doesn't matter.

  • Apple
  • Banana
  • Cherry

3. Nested Lists

A nested list is a list inside another list. It can be ordered or unordered.

  • Fruits
    • Apple
    • Banana
  • Vegetables
    • Carrot
    • Spinach

4. Ordered List Inside an Unordered List

An ordered list can be placed inside an unordered list item for different organization styles.

  • Shopping
    1. Milk
    2. Eggs

5. Definition List

A definition list displays terms and their definitions. It consists of 'dt' for terms and 'dd' for definitions.
